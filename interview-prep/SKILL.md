# Interview Prep

## 流程

1. 读取岗位画像、最终简历和匹配诊断。
2. 生成三类题：岗位基础题、项目追问题、短板补强题。
3. 每题附考察点和答题要点；项目题必须绑定简历中的具体经历。
4. 对风险项生成追问，帮助用户提前补证据或收敛表述。

## 默认输出

| 类型 | 数量 | 说明 |
|------|------|------|
| 岗位基础题 | 5 道 | 围绕岗位核心技能 |
| 项目追问题 | 5 道 | 绑定简历中的具体项目 |
| 短板补强题 | 3 道 | 围绕匹配诊断中的真实缺口 |
| 面试准备建议 | 1 段 | 优先级排序 |

## 题目格式

```markdown
### Q{序号}：{题目}

**考察点：** {考察内容}

**答题要点：**
- {要点1}
- {要点2}
- {要点3}

**可结合项目：** {简历中的具体项目/模块}

**避免的夸大表述：** {风险提示}
```

## 边界

- 不堆砌题库大全
- 不生成与简历无关的项目追问
- 不把用户不会的短板包装成已掌握能力
