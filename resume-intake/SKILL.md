# Resume Intake

## 流程

1. 读取用户提供的简历、项目文档、作品链接和会话材料。
2. 提取基本信息、教育背景、技能栈、项目经历、实习经历、竞赛/科研、获奖证书。
3. 为每条经历记录证据来源：文件名、段落摘要、链接或用户原句。
4. 按目标岗位标记相关度：高、中、低、待确认。
5. 输出 `CandidateProfile` 和 `uncertain_items`。

## 解析原则

- 原文有歧义时保留原句，不擅自补全。
- 联系方式、学校、公司、时间、数字成果必须来自材料。
- 对“熟悉/了解/掌握”等技能词，结合项目证据判断可信度。
