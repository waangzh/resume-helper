# Learning Plan

## 流程

1. 读取匹配诊断中的真实缺口和面试风险。
2. 按优先级拆成短期可补、需要项目证明、暂不建议写入简历三类。
3. 给出 7 天或 14 天行动计划，包含知识点、练习任务、产出物。
4. 推荐资料时优先给官方文档、经典教程或可搜索关键词。

## 输出要求

- 每个短板都要有明确下一步动作。
- 学习建议要能反哺简历或面试准备。
- 给具体链接时必须确认来源可访问；无法确认时给搜索关键词而非伪造链接。

## 边界

- 不把学习计划写成用户已有能力。
- 不推荐过多资料，默认每个短板 1-3 个高价值资源。

## 资源

需要资料推荐策略、链接边界和默认时间盒时读取 `references/resource-policy.md`。
