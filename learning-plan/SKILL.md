# Learning Plan

## 流程

1. 读取匹配诊断中的真实缺口和面试风险。
2. 按优先级拆成短期可补、需要项目证明、暂不建议写入简历三类。
3. 给出 7 天或 14 天行动计划，包含知识点、练习任务、产出物。
4. 推荐资料时优先给官方文档、经典教程或可搜索关键词。

## 缺口分类

| 类型 | 时间要求 | 处理方式 |
|------|----------|----------|
| 短期可补 | 7-14 天 | 安排学习任务 |
| 需要项目证明 | 14+ 天 | 安排补项目行动 |
| 暂不建议写入简历 | - | 标记为后续目标 |

## 行动计划格式

```markdown
### {短板名称}

**缺口类型：** {短期可补/需要项目证明}

**学习目标：** {具体可衡量的目标}

**时间安排：** {天数}天

**知识点：**
- {知识点1}
- {知识点2}

**练习任务：**
- {任务1}
- {任务2}

**产出物：** {可验证的成果}

**推荐资料：**
- {资料1}（官方文档）
- {资料2}（搜索关键词）
```

## 输出要求

- 每个短板都要有明确下一步动作
- 学习建议要能反哺简历或面试准备
- 给具体链接时必须确认来源可访问；无法确认时给搜索关键词

## 边界

- 不把学习计划写成用户已有能力
- 不推荐过多资料，每个短板 1-3 个高价值资源
- 时间盒默认 7 天（面试前快速补强）或 14 天（补项目证据）
