# 岗位搜索与合规

## 搜索策略

| 步骤 | 操作 | 说明 |
|------|------|------|
| 1 | 组合关键词 | 岗位名 + 实习/校招 + 城市 + 核心技术 |
| 2 | 选择平台 | 优先用户指定平台，否则使用公开可访问渠道 |
| 3 | 筛选结果 | 只保留有明确岗位信息的详情页 |
| 4 | 记录信息 | 标题、公司、地点、平台、链接、检索日期 |

## 支持平台

| 平台 | 类型 | 注意事项 |
|------|------|----------|
| BOSS 直聘 | 综合招聘 | 可能需要登录，遇到验证码停下 |
| 拉勾 | 互联网 | 可能需要登录 |
| 猎聘 | 中高端 | 可能需要登录 |
| 智联招聘 | 综合招聘 | - |
| 前程无忧 | 综合招聘 | - |
| 牛客 | 校招/实习 | - |
| 实习僧 | 实习 | - |
| LinkedIn | 国际/高端 | 需登录 |
| 公司 ATS | 官方直投 | 适合精准投递 |

## 链接要求

**必须：** 岗位详情页链接，能直接打开到具体岗位

**禁止：**
- 公司官网首页
- 招聘首页
- 搜索结果页
- 聚合页
- 无法定位岗位的短链

## Excel 清单字段

| 字段 | 必填 | 格式要求 |
|------|------|----------|
| 岗位名称 | 是 | 文本 |
| 公司名称 | 是 | 文本 |
| 城市 | 是 | 文本 |
| 平台 | 是 | 文本 |
| 学历要求 | 是 | 文本 |
| 技术关键词 | 是 | 文本，逗号分隔 |
| 岗位详情页 | 是 | 可点击超链接 |
| 来源 | 是 | 文本 |
| 检索日期 | 是 | YYYY-MM-DD |
| 推荐理由 | 是 | 文本 |
| 投递要求 | 否 | 文本 |
| 状态 | 是 | 见状态枚举 |
| 备注 | 否 | 文本 |

## 状态枚举

| 状态 | 说明 | 下一步 |
|------|------|--------|
| 待用户确认 | 初始状态 | 等待用户选择 |
| 用户已选择 | 用户确认投递 | 执行投递 |
| 已打开 | 已打开岗位页面 | 填写表单 |
| 已投递 | 投递成功 | 记录结果 |
| 投递受阻 | 遇到障碍 | 处理问题 |
| 需用户登录 | 需要登录 | 等待用户操作 |
| 已跳过 | 用户跳过 | 不处理 |

## 禁止事项

- 未经用户确认自动投递
- 保存、索要或复用用户账号密码
- 编造岗位、公司或链接
- 将广告或培训信息当成招聘岗位

## 投递记录格式

| 字段 | 内容 |
|------|------|
| 岗位 | 岗位名称、公司、平台、详情页 |
| 用户确认 | 用户选择和确认投递的原句或时间 |
| 使用材料 | PDF 简历、HTML 预览、投递说明 |
| 已执行动作 | 打开页面、登录等待、填表、上传、提交 |
| 结果 | 已投递、停在最终确认、需登录、失败 |
| 证据 | 页面标题、时间、错误信息或下一步 |

## 最终提交规则

- 用户已明确授权”提交/投递”时，可以点击最终提交
- 平台提示付费、隐私授权、职位不一致、岗位已下线、材料不可确认时，**不提交**，停在确认点并说明原因
