# 软技能标签体系

> 通用岗位能力评估与表达的基础标签库。

---

## 核心能力标签

### 沟通表达

| 标签 | 定义 | 简历表达示例 |
|------|------|--------------|
| 口头表达 | 清晰、有条理地口头传达信息 | 负责项目汇报，向10人团队演示技术方案 |
| 书面表达 | 准确、规范地撰写文档、报告 | 撰写活动策划案3份，获学院审批通过 |
| 倾听理解 | 准确理解他人需求和反馈 | 收集用户需求20+条，整理为产品需求文档 |
| 说服影响 | 通过沟通影响他人决策 | 说服3家企业成为活动赞助商 |

### 团队协作

| 标签 | 定义 | 简历表达示例 |
|------|------|--------------|
| 协作配合 | 与他人配合完成任务 | 与5人团队协作完成校园活动策划 |
| 冲突处理 | 处理团队内部矛盾和分歧 | 协调部门间分工分歧，推动项目按期完成 |
| 跨部门协调 | 协调不同部门/团队资源 | 协调宣传、外联、后勤3个部门完成活动执行 |
| 支持他人 | 主动帮助团队成员 | 帮助3名新成员快速适应团队工作 |

### 组织协调

| 标签 | 定义 | 简历表达示例 |
|------|------|--------------|
| 活动策划 | 制定活动方案并组织执行 | 策划校园歌手大赛，统筹20人团队完成执行 |
| 资源整合 | 整合人力、物力、财力资源 | 整合校内外资源，筹办200人规模讲座 |
| 时间管理 | 合理安排任务优先级和时间节点 | 同时管理3个项目，均按期交付 |
| 流程优化 | 改进工作流程提升效率 | 优化活动报名流程，效率提升50% |

### 领导力

| 标签 | 定义 | 简历表达示例 |
|------|------|--------------|
| 团队管理 | 带领团队完成目标 | 带领15人团队完成年度工作计划 |
| 决策判断 | 在复杂情况下做出合理决策 | 活动当天突发状况，快速决策调整方案 |
| 培养他人 | 帮助团队成员成长 | 培养2名干事成长为部门骨干 |
| 目标达成 | 带领团队达成既定目标 | 带领团队完成招生目标120% |

### 学习能力

| 标签 | 定义 | 简历表达示例 |
|------|------|--------------|
| 快速学习 | 短时间内掌握新知识/技能 | 1周内掌握新媒体运营工具，独立完成推送 |
| 自主学习 | 主动获取知识，不需要外部推动 | 自学Python，完成数据分析课程项目 |
| 知识迁移 | 将已有知识应用到新领域 | 将课堂营销理论应用于社团推广实践 |
| 持续学习 | 保持学习习惯，不断更新知识 | 保持每周阅读2篇行业文章的习惯 |

### 执行力

| 标签 | 定义 | 简历表达示例 |
|------|------|--------------|
| 任务落实 | 将计划转化为具体行动 | 落实活动执行方案，协调场地、物资、人员 |
| 结果导向 | 以结果为目标推动工作 | 为拉取赞助，拜访企业30家，成功8家 |
| 抗压执行 | 在压力下保持工作效率 | 期末期间同时完成学业和活动筹备 |
| 细节把控 | 关注执行过程中的关键细节 | 活动现场负责签到，0差错完成200人签到 |

### 抗压能力

| 标签 | 定义 | 简历表达示例 |
|------|------|--------------|
| 压力适应 | 在压力环境下保持稳定表现 | 兼职期间应对客流高峰，保持服务质量 |
| 情绪管理 | 在困难情况下控制情绪 | 处理客户投诉10+次，保持专业态度 |
| 多任务处理 | 同时处理多项任务 | 同时处理课程、社团、兼职三方面任务 |
| 挫折应对 | 面对失败或挫折后快速恢复 | 竞赛失利后总结经验，下次获得更好成绩 |

### 问题解决

| 标签 | 定义 | 简历表达示例 |
|------|------|--------------|
| 问题分析 | 准确识别问题根源 | 分析活动参与率低的原因，提出改进方案 |
| 方案设计 | 针对问题提出解决方案 | 设计会员管理流程，解决信息混乱问题 |
| 应急处理 | 突发情况下的快速反应 | 活动当天设备故障，1小时内协调备用方案 |
| 创新解决 | 用创新方式解决问题 | 设计线上报名系统，替代纸质报名提升效率 |

## 能力等级定义

| 等级 | 名称 | 定义 | 证据要求 |
|------|------|------|----------|
| 1 | 了解 | 知道基本概念和方法 | 课程学习、阅读了解 |
| 2 | 运用 | 能够在指导下运用 | 课程项目、简单实践 |
| 3 | 熟练 | 能够独立完成相关工作 | 实习经历、项目实践 |
| 4 | 精通 | 能够指导他人或处理复杂情况 | 领导经历、复杂项目 |
| 5 | 专家 | 能够创新方法或解决疑难问题 | 专业成果、行业认可 |

## 能力标签使用方式

### 1. 经历萃取时

从用户经历中识别体现的能力标签：

```markdown
| 经历 | 体现的能力标签 | 证据 |
|------|----------------|------|
| 学生会外联部干事 | 沟通表达、抗压能力、结果导向 | 电话沟通企业30家，成功拉取赞助8000元 |
| 校园歌手大赛组织者 | 组织协调、领导力、问题解决 | 统筹20人团队，活动参与800人 |
| 家教兼职 | 沟通表达、学习能力、执行力 | 辅导学生成绩提升25分 |
```

### 2. 经历筛选时

根据目标岗位所需能力筛选经历：

```markdown
目标岗位：管培生
核心能力：学习能力、沟通表达、团队协作、领导力

经历匹配：
- 学生会主席（领导力、组织协调）→ 高匹配，突出
- 销售兼职（沟通表达、抗压能力）→ 高匹配，保留
- 编程项目（问题解决）→ 中匹配，保留
- 无关课程作业 → 低匹配，不写入
```

### 3. 简历改写时

在 bullet 中体现能力标签：

```markdown
# 原始描述
在学生会做过干事，负责外联工作

# 能力导向改写
担任学生会外联部干事，负责校企合作赞助招商（组织协调）
筛选目标企业50家，电话沟通30家，实地拜访8家（执行力、抗压能力）
成功拉取赞助8000元，覆盖活动成本60%（结果导向）
```

### 4. 匹配诊断时

对照岗位能力需求分析简历：

```markdown
岗位需求：沟通表达、团队协作、学习能力

简历分析：
✓ 沟通表达：有外联经历，有证据支撑
✓ 团队协作：有社团经历，有协作描述
⚠ 学习能力：有专业成绩，但缺少自学/跨领域证据
```
