# 技术岗基线

## Java 后端

- 核心：Java、Spring Boot、MyBatis 或 JPA、REST API、MySQL。
- 加分：Redis、消息队列、Linux、Docker、简单性能优化。
- 面试：Java 基础、集合、多线程、JVM、数据库索引、事务、缓存、项目追问。

## 前端开发

- 核心：HTML、CSS、JavaScript、TypeScript、Vue 或 React、组件化、接口联调。
- 加分：构建工具、状态管理、性能优化、可访问性、移动端适配。
- 面试：JS 基础、浏览器、异步、框架原理、组件设计、工程化。

## 测试开发

- 核心：测试用例设计、缺陷管理、接口测试、自动化测试、Python/Java、SQL。
- 加分：性能测试、CI/CD、测试平台、Mock、Linux。
- 面试：测试方法、接口测试、自动化框架、数据库、问题定位。

## 算法实习

- 核心：数据结构、算法、Python/C++、机器学习基础、实验复现。
- 加分：深度学习框架、论文阅读、Kaggle/竞赛、模型部署基础。
- 面试：算法题、模型基础、项目实验设计、指标分析。
