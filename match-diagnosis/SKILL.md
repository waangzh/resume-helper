# Match Diagnosis

## 流程

1. 对照岗位画像检查简历中已体现的核心能力。
2. 区分表达不足和真实缺失：前者可改写，后者需要学习或补项目。
3. 标记面试风险项：写入简历但证据弱、容易被追问穿透的内容。
4. 输出匹配诊断，供面试题和学习计划使用。

## 输出结构

- 已命中能力
- 待优化表达
- 真实能力缺口
- 面试风险
- 下一步动作

## 边界

- 不用虚假百分比分数制造精确感；可用高/中/低或 1-5 档。
- 建议必须可执行，例如补一个案例、准备一个问题、完善一个指标。
